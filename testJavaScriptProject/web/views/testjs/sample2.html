  2. window 객체의 후손인 document 객체가 제공하는 write() 또는 writeln() 메소드로 body에 출력하는 방법
  3. 태그 객체의 innerHTML 속성을 사용해서 태그안에 값으로 출력하는 방법
  4. 개발자 도구의 console 뷰에 출력하는 방법
