자바스크립트 작성 위치와 작성 방법

inline 방식 : 시작태그에 on이벤트라는 속성을 추가하고 자바스크립트 소스를 바로 작성할 수 있다.
on이벤트 속성은 자바스크립트가 제공하는 속성이며, 모든 태그에 사용할 수 있다.

자바스크립트 소스를 script 태그 안에 함수(function) 단위로 작성하는 경우

script 태그의 작성 위치는 head, body, html밖 어디든 작성 가능하며, 한 문서 안에 여러번 사용 가능하다. html 태그에서 자바스크립트 소스를 연결 실행시키는 방법은 시작태그에 on이벤트 속성 추가하고, 값으로 실행할 자바스크립트 함수명을 기입하면 됨

external : 별도의 자바스크립트 소스 파일(*.js)을 작성해 두고 필요한 웹문서에서 가져다 사용하는 방식
